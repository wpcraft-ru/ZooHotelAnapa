# Зоогостиница Анапа — лендинг

Забота о вашем питомце, пока вы в отъезде. Уютная передержка для собак и кошек без клеток.

## Контакты

- 📍 ул. Фрунзе, 64/2, с. Супсех, Анапа
- 📞 +7 (918) 648-14-44
- ⭐ 5.0 на Яндекс.Картах (92+ оценки)

## Быстрый старт

```bash
make dev
```

Открыть http://localhost:4321

## Структура проекта

- `DESIGN.md` — дизайн-токены и система
- `MARKETING.md` — маркетинговая стратегия, SEO, портрет клиента
- `.agents/CONTEXT.md` — контекст проекта
- `docs/rfc/` — page design RFC файлы
- `docs/prototypes/zoohotel-anapa/` — HTML-прототип на AstroJS + DaisyUI
